# ABS

![ABS Block](../../images/ABS.png)

**Category:** OE Logic > Math

**Description:** Removes the minus sign from your number.

**Input types:**  
- number  
- vector (both inputs must be the same type)

**Output types:**  
- number  
- vector (same type as inputs)

**Example:**  

- -5 -> 5  
- (-2,5,-7) -> (2,5,7)
