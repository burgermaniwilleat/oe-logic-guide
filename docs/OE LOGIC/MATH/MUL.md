# MUL (Multiplication)

![MUL Block](../../images/MUL.png)

**Category:** OE Logic > Math  

**Description:** Multiplies both values together.  

**Input types:**  
- number  
- vector (both inputs must be the same type)  

**Output types:**  
- number  
- vector (same type as inputs)  

**Examples:**  
- `5 * 5 = 25`  
- `9 * -12 = -108`  
- `-8 * 0 = 0`  
- `(10, 20, -13) * (5, -3, -7) = (50, -60, 91)`
