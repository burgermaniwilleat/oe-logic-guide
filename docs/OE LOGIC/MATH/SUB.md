# SUB (Subtraction)

![SUB Block](../../images/SUB.png)

**Category:** OE Logic > Math  

**Description:** Subtracs the second value from the first.  

**Input types:**  
- number  
- vector (both inputs must be the same type)  

**Output types:**  
- number  
- vector (same type as inputs)  

**Examples:**  
- `3 - 4 = -1`  
- `(10, 20, 13) - (5, 3, 7) = (5, 17, 6)`  
- `-5 - 3 = -8`
