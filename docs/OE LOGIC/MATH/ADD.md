# ADD

![ADD Block](../../images/ADD.png)

**Category:** OE Logic > Math  

**Description:** Adds values together.  

**Input types:**  
- number  
- vector (both inputs must be the same type)  

**Output types:**  
- number  
- vector (same type as inputs)  

**Examples:**  
- `3 + 4 = 7`  
- `(10, 20, 13) + (5, 3, 7) = (15, 23, 20)`  
- `-5 + 3 = 2`
