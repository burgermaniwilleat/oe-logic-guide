# FLOOR

![FLOOR Block](../../images/FLOOR.png)

**Category:** OE Logic > Math  

**Description:** Rounds input numbers down.  

**Input types:**  
- number  
- vector (both inputs must be the same type)  

**Output types:**  
- number  
- vector (same type as inputs)  

**Examples:**  
- `-5.43 → -6`  
- `3.5 → 3`  
- `(10.4, -20.6, 13.5) → (10, -21, 13)`
