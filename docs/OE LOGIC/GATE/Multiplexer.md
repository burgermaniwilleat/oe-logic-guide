# Multiplexer

![Multiplexer Block](../../images/Multiplexer.png)

**Category:** OE Logic > Gates

**Description:** Outputs values depending on 'State' input.

**Input types:**  
- Universal 

**State types:**
- bool
- number
- byte

**Output types:**  
- Universal

**Simulation:**  

**Number/Byte State:**

|  State  | Output |
|---------|--------|
|   ≤ 0   | False input |
|   ≥ 1   | True input |



**Bool State:**

|  State  | Output |
|---------|--------|
|   False | False input |
|   True  | True input |
